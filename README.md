\# ✍️ PenWriter – Handwriting Animation Generator  

\*A lightweight browser tool that turns any text into a smooth handwritten animation with a moving pen.\*



---



\## 🚀 Overview



\*\*PenWriter\*\* allows you to paste any text and watch it appear on screen as if it’s being written by hand — letter by letter — with a pen following the text in real time.



No backend.  

No installation.  

Everything runs \*\*inside your browser\*\* using HTML5 Canvas.



---



\## 🎯 Features



\- ✔ \*\*Paste any text\*\* — LinkedIn posts, quotes, poems, notes, anything  

\- ✔ \*\*Smooth handwriting animation\*\* (characters appear naturally)  

\- ✔ \*\*Pen follows the writing\*\* for realistic effect  

\- ✔ \*\*Auto-scrolling\*\* as writing moves down the canvas  

\- ✔ \*\*Supports line breaks, paragraphs, emojis\*\*  

\- ✔ \*\*Standalone HTML\*\* — works instantly  

\- ✔ \*\*Zero dependencies\*\* (no FFmpeg, no server)  

\- ✔ \*\*Fully customizable\*\* (font, speed, pen size, etc.)
